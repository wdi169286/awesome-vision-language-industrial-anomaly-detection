# Contributing

Thank you for helping maintain this list.

## Adding a Paper

Please include:

- paper title
- year and venue or source
- paper link
- code link if available
- taxonomy role
- inspection setting
- datasets and metrics
- a short protocol note

Use the taxonomy roles below.

- semantic-prior
- cross-modal-local-evidence
- scarce-sample-calibration
- auxiliary-foundation-prior
- reasoning-level-inspection
- benchmark-or-dataset
- future-reference

## Quality Rules

- Add only verified papers, official project pages, or stable preprints.
- Do not report a metric without its dataset, task level, and protocol setting.
- Do not merge zero-shot, few-shot, online, and reasoning protocols into one ranking.
- Prefer source-paper numbers. If a number comes from another comparison table, name the source.
- Keep titles in title case and preserve the official method name.

## Pull Request Checklist

- [ ] The paper is placed under the correct evidence role.
- [ ] The protocol setting is clear.
- [ ] Links are verified.
- [ ] Dataset and metric names are explicit.
- [ ] The entry is also added to `data/papers.yml` when appropriate.
